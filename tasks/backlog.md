# Бэклог задач

Этот документ содержит список задач для будущих спринтов.

## Stem Studio V2
- [ ] **Реальная загрузка данных:** Заменить фиктивные данные о стемах на реальную загрузку с бэкенда.
- [ ] **Визуализация аудио:** Заменить плейсхолдеры для звуковых волн на реальную визуализацию с помощью Web Audio API.
- [ ] **Экспорт в WAV:** Добавить функционал конвертации и экспорта микса в формат WAV.
- [ ] **Интеграция регенерации:** Подключить кнопку регенерации к соответствующей Edge-функции.
- [ ] **Улучшение UX текста:** Сделать текст песни интерактивным, чтобы можно было кликать на слово для перемотки.
- [ ] **Сохранение состояния:** Сохранять состояние студии (громкость, соло/мьют) между сессиями.

## Общие улучшения
- [ ] **Тестовое покрытие:** Написать unit-тесты для новых компонентов и хуков "Студии".
- [ ] **Оптимизация производительности:** Провести аудит производительности "Студии" при работе с большим количеством стемов.
- [ ] **Документация API:** Описать API для получения данных о стемах в `docs/api`.
