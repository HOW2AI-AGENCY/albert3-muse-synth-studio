# Load Test Plan (1000+ RPS)

## Сценарии
- Прогрев (warm-up), ступенчатый рост (ramp-up), плато, деградация.
- Операции: генерация запроса, получение статуса, загрузка ресурсов.

## Конфигурация JMeter
- Thread Group: до 1000+ параллельных потоков.
- Ramp-up: 5–10 мин до целевой нагрузки.
- Таймауты/ретраи: включены.

## Метрики
| Метрика | Цель |
|---|---|
| p50 latency | < 500ms |
| p95 latency | < 1500ms |
| p99 latency | < 2500ms |
| Error Rate | < 1% |
| Throughput | ≥ требуемой нагрузки |

## Отчётность
- HTML отчеты JMeter + CSV артефакты.
- Графики: latency/throughput/error rate.
