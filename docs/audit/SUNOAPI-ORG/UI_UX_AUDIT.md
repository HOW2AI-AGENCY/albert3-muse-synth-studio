# UI/UX Аудит (Web/Mobile)

## Обзор
- Платформы: Web (Chrome/Firefox/Safari), мобильные браузеры Android/iOS.
- Фокус: Mobile First, интуитивность, эффективность, удовлетворённость.

## Методология
- Heuristic Evaluation (Нильсен), Cognitive Walkthrough, анализ пользовательских потоков.
- Тестирование сценариев: генерация треков, загрузка обложек, навигация Workspace.
- Брейкпоинты: 320×568, 360×640, 390×844, 768×1024, 1440×900.

## Доступность (WCAG 2.1 AA)
- Контраст, фокус, клавиатурная навигация, ARIA, альтернативные тексты.
- Авто-проверки: Axe/Playwright Axe; ручные проверки.

## Соответствие HIG/Material
- Проверка паттернов компонентов, жестов, размеров touch targets (≥44px).
