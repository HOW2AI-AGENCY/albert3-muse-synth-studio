# Дизайн-система "MusicVerse"

**Версия:** 1.0.0
**Статус:** В разработке

---

## 1. Философия

"MusicVerse" — это система, построенная на принципах минимализма, элегантности и функциональности. Наша цель — создать интуитивно понятный, отзывчивый и эмоционально привлекательный интерфейс для музыкантов и продюсеров. Дизайн должен быть незаметным, позволяя пользователю сосредоточиться на творчестве.

- **Минимализм:** Только необходимые элементы. Никакого визуального шума.
- **Элегантность:** Чистые линии, выверенная типографика и гармоничная цветовая палитра.
- **Функциональность:** Интерфейс должен быть не только красивым, но и быстрым, и эффективным.
- **Mobile-First:** Проектирование от малого экрана к большому, гарантируя идеальный опыт на любом устройстве.

## 3. Типографика

**Шрифт:** Inter (уже используется в проекте, что хорошо). Он отлично подходит благодаря своей нейтральности и читаемости.

| Элемент | Размер (font-size) | Насыщенность (font-weight) | Межстрочный интервал (line-height) |
| :--- | :--- | :--- | :--- |
| **Heading 1 (`h1`)** | 2rem (32px) | 700 (Bold) | 2.5rem |
| **Heading 2 (`h2`)** | 1.5rem (24px) | 700 (Bold) | 2rem |
| **Heading 3 (`h3`)** | 1.25rem (20px) | 600 (SemiBold) | 1.75rem |
| **Body (p)** | 0.9375rem (15px) | 400 (Regular) | 1.5rem |
| **Small** | 0.8125rem (13px) | 400 (Regular) | 1.25rem |
| **Button** | 0.9375rem (15px) | 500 (Medium) | 1.25rem |


## 4. Сетка и отступы

Используем систему отступов, кратную 4px.

- `p-1` -> 4px
- `p-2` -> 8px
- `p-3` -> 12px
- `p-4` -> 16px
- `p-6` -> 24px
- `p-8` -> 32px

**Основной контейнер:**
- `container-normal`: `max-width: 1600px; padding: 0 1rem;`
- `container-inline`: без `max-width`, но с `padding`.

## 5. Иконография

**Стиль:** Line icons. Толщина линии 1.5px.
**Библиотека:** `lucide-react`. Необходимо провести ревизию и убедиться, что все иконки импортируются из `src/utils/iconImports.ts` для консистентности.

## 6. Компоненты (Аудит и План Улучшений)

### Button
- **Текущее состояние:** Стандартный `shadcn/ui` button.
- **Улучшения:**
    - Уменьшить `padding` для более компактного вида (`h-9` вместо `h-10` для стандартного размера).
    - Добавить тонкую, почти незаметную тень при `hover` и `focus`.
    - Добавить микро-анимацию: легкое "нажатие" (scale down) при `active`.

### Card

- **Улучшения:**
    - Использовать градиентную рамку для выделенных или активных карточек.
    - Добавить анимацию `hover`: легкое поднятие (translateY) и свечение.

### Input / Select
- **Текущее состояние:** Стандартные.
- **Улучшения:**
    - Сделать фон инпутов чуть темнее фона карточки для лучшего контраста.
    - Добавить плавную анимацию `focus` на рамке.
